# Simple_MCQ_Quiz_App
To design and implement a simple MCQ quiz application that displays questions, accepts user answers, keeps track of correct and wrong answers, and calculates the final score.
This project is a simple MCQ quiz application. The user is asked 20 multiple-choice questions.
For each correct answer, the score is increased by 1. Wrong answers are counted separately.
At the end, the final score and the number of wrong answers are displayed
